三月总结

发表于 2017-07-27 | 分类于 生活 | | 浏览

不要被自己设置的穹顶限制
Under The Dome

学习

课内

大一下过的很快，上一篇文章5月15号属于下学期一半的一半。期中考试差不多要忘记，期末考试还没那么近的时候。大一下不是那么的担心成绩，不挂科还是没问题的。上了大学之后对于学习的态度发生了几次转变，最近的就在这时（5月15号）。

不管怎么样，结果是没挂科。名次也进不了，进了前二十（滑稽）。但实际上来说感觉一般，物理老师抬了一手。

课外

5月15号附近去了实验室，具体也不多说了，但确实是对于我的整个学习过程有较大的影响。至于能力是否提升了，有待讨论。

生活

关于生活，并没有什么想说的。5月15号的上一篇文章，看似巧合。只是现在留校了，还是没去学车。有一点烦，就一点。这应该算生活了。

读后感

看了什么书？
好像就前几天去看的《解忧杂货店》，一直听说合肥的三孝口新华书店挺好，作为一个自认为的读书人当然要去看看了。关于书店感觉确实挺好，主要是有一种包容的气息，人们坐在地上，书架旁看着书。有几岁的小孩，也有二十多岁的大学生；
不得不说这本书确实很好看，一共有四章？，第一章就击中了我的内心，每个人都有着自己的不为人知的一面，或邪恶，或自私，或懦弱。看似矛盾的背后其实是在寻找着借口
，说服别人、说服自己。有时候没有必要纠结于该怎么走，只要大胆的走，做出目前看来最正确的选择就不会后悔。

留校已经将近二十天了，学习就那样，兼职倒是挣了不少，但最近的生活作息及别的一些实在不能让我满意。
17届的新生已经蠢蠢欲动了，配得上被人叫一声学长吗？
